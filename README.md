\# nifty50-ml-pipeline



End-to-end Machine Learning pipeline for Nifty 50 minute-level data (2015–2025).  

Includes data preparation, EDA, feature engineering, PCA, clustering, regression/classification models, LSTM forecasting, and model evaluation.



\## Project Structure

\- data/ → raw dataset (ignored from Git)

\- notebooks/ → Jupyter notebooks for EDA \& modelling

\- src/ → Python scripts for preprocessing, merging, modelling

\- models/ → saved models (ignored)

\- reports/ → visuals, PDFs, notes



\## How to Start

1\. Create a virtual environment

2\. Install requirements using `pip install -r requirements.txt`

3\. Place raw data files in `data/` folder

4\. Run scripts from `src/` or explore notebooks in `notebooks/`
